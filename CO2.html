CO2 Emissions Data: Interactive Dashboard

An exploratory analysis project on Python using CO2 databases would involve analyzing yearly CO2 emissions on metric tons from 2000 to 2018.

The analysis focused on identifying patterns and trends in the data, such as changes in global emissions over time and regional variations in emissions.

Additionally, an interactive dashboard was created using panel and hvplots to provide an engaging and user-friendly way to explore the data.

Analysis Tool(s): Python.

Database Source: CO2 Emissions _ 1960 - 2018

Code Source: Pyhon CO2 Emissions Code

Explored Files: CO2 emissions by country since 2000, countries with the maximum and minimum CO2 emissions and the evolution of CO2 emissions on the world over the years.
