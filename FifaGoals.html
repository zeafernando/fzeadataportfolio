FIFA World Cup Goals Scored Data: Exploration

An exploratory analysis project using SQL on the FIFA World Cup Goals Scored database which contains information about all the goals scored since the 1930 edition up until 2022.

The available information includes the edition played, team, type of goal, the minute in which the goal was scored, and the name of the player.

The project involved querying the database to extract information about the number of goals scored in each edition, the teams that scored the most goals, and the players with the highest goal-scoring records.

Analysis Tool(s): SQL.

Database Source: FIFA World Cup All Goals 1930-2022

Code Source: SQL FIFA World Cup Goals Query

Explored Files: Editions of the FIFA World Cup since 1930, editions played by country since 1930, total goals, total own goals, total penalties, total goals per tournament, country, player, match minute, etc.
