COVID-19 Data: Exploration

An exploratory analysis project on SQL using WHO databases about COVID-19 cases and vaccinations. Insights about the pandemic's impact on different countries and regions were obtained.

The project queries databases to extract information on cases, deaths, vaccinations, and demographic data. Patterns and trends in the data would be identified, and visualizations would be used to communicate the findings.

Analysis Tool(s): SQL.

Database Source: WHO COVID-19

Code Source: SQL COVID-19 Query

Explored Files: Total COVID 19 cases by country, total COVID 19 deaths by country, percentage regarding deaths, global numbers by country, comparison of people fully vaccinated vs people with only one dose & percentage of vaccinations
